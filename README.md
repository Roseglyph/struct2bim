# Struct2BIM Synthetic Lab

Struct2BIM is a local portfolio project for generating synthetic structural-drawing datasets, exporting exact computer-vision annotations, normalizing structural geometry, and producing BIM/IFC artifacts.

## Current status

The repository is being initialized locally. The synthetic generator, annotation pipeline, IFC showcase, and local training workflow will be implemented in reviewable stages.

No trained model weights or detector performance results are included at this stage.
